## This project is created to practice Selenium WebDriver concepts using Java.
It focuses on handling parent to child window navigation, multiple tabs, and link interactions using real browser automation.

### Tech Stack
- Java
- Selenium WebDriver
- Maven
- Eclipse
- Git & GitHub

### Features
- Location detection automation
- Category navigation
- Multiple product add-to-cart
- Quantity handling using loops
- Selenium WebDriver with Java
- Parent to Child window handling
- Opening multiple links in different browser tabs
- Switching between tabs/windows
- Clicking all links one by one
- Basic synchronization using WebDriverWait
- XPath & CSS Selectors
- Real-time browser interaction
- Login validation and fill form 

### Selenium Concepts Implemented
 - getWindowHandles()
 - switchTo().window()
 - Parent and Child window identification
 - Looping through multiple windows
 - Opening links in new tabs
 - Handling dynamic elements

### How to Run
1. Clone repository
2. Open in Eclipse
3. Run Addtocart.java
